# NAMI

**Navigating And Management Intelligence**

NAMI is a desktop AI agent for Windows that performs visible, human-like interaction with the operating system.

Rather than silently manipulating files through APIs, NAMI navigates Windows applications as a human would whenever practical.

---

## Technology Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS (planned)
- Zustand (planned)

### Backend

- Python 3.12
